## Agent Multibagger

Agent Multibagger is an interactive Next.js + Tailwind web application that acts as an equity research co-pilot for Indian markets. Feed the agent your macro view, risk appetite and secular trend focus and it triangulates a curated list of potential multibagger candidates using a multi-factor framework covering fundamentals, growth velocity, innovation, catalysts, valuation sanity and risk control.

### Run locally

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to explore the agent dashboard.

### Feature highlights

- Macro narrative engine summarising GDP/Inflation/Rate regime vs megatrend focus.
- Investor blueprint controls for risk tolerance, horizon, capital and catalyst bias.
- Five-step scoring grid per company with rationale bullets and catalyst tracker.
- Portfolio blueprint with suggested allocation and thesis notes in INR lakhs.
- Risk dashboard with systemic/company-specific callouts and mitigation stack.
- Radar watchlist built from the remaining high-growth universe for future study.
